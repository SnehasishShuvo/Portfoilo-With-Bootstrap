HTML Tags and Their Descriptions

Tag Description
<!DOCTYPE> Defines the document type and version of HTML.
<html> Root element of an HTML document.
<head> Contains metadata about the document, like title, styles, and scripts.
<title> Sets the title of the document, shown in the browser tab.
<meta> Specifies metadata like charset, viewport, or descriptions.
<link> Links external resources like CSS files.
<style> Defines internal CSS styles.
<script> Adds JavaScript code to the document.
<body> Contains the visible content of the HTML page.
<h1> to <h6> Defines headings, from largest (<h1>) to smallest (<h6>).
<p> Defines a paragraph.
<a> Creates hyperlinks to other pages or sections.
<img> Embeds images into the document.
<ul> and <li> Creates an unordered list (<ul>) with list items (<li>).
<ol> and <li> Creates an ordered list (<ol>) with list items (<li>).
<table> Creates a table for displaying data.
<tr>, <td>, <th> Defines table rows (<tr>), data cells (<td>), and header cells (<th>).
<div> Defines a container or section in the document.
<span> Defines inline elements for styling or grouping text.
<br> Inserts a line break.
<hr> Inserts a horizontal rule (line).
<form> Defines a form for user input.
<input> Specifies an input field for forms.
<button> Defines a clickable button.
<label> Associates text with a form control.
<iframe> Embeds another webpage into the document.
<strong> and <b> Makes text bold; <strong> adds importance.
<em> and <i> Makes text italic; <em> emphasizes importance.
<footer> Defines the footer of a page or section.
<header> Defines the header of a page or section.
<nav> Defines a section for navigation links.
<section> Defines a standalone section in a document.
<article> Defines independent, self-contained content.
